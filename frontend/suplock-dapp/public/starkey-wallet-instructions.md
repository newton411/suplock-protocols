# Starkey Wallet Integration (Supra L1)

## Installation

1. **Install the Starkey Wallet Extension**
   - Visit [Supra Starkey Wallet](https://supra.com/wallets/starkey) or search "Starkey Wallet" in your browser extension store.
   - Click "Add to Chrome" (or your browser) and follow the prompts.

2. **Create or Import a Wallet**
   - Open the extension and follow the onboarding steps to create a new wallet or import an existing one.

3. **Connect to Supra L1**
   - Ensure the network is set to "Supra L1 Testnet" or "Supra L1 Mainnet" as appropriate.

## Usage in SUPLOCK Dapp

- When you click **Connect Wallet**, SUPLOCK will detect Starkey Wallet automatically if installed.
- Only Starkey Wallet is supported for Supra L1. No other wallet options are available.
- If Starkey Wallet is not installed, you will be prompted to install it.
